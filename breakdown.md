# Baristamatic 
A command line automatic coffee dispensing machine simulator

##### Main
 - run restocking function to make sure initial inventory is always stocked when app is started
 - display welcome message
 - display inventory
 - display menu
 - prompt user to order drink, restock inventory or quit app
 - run appropriate function based on users selection if applicable
 - display appropriate message

#### Display
 - function to map inventory
 - helper function to display each ingredient and it's stock
 - function to map menu 
 - helper function to display menu number, beverage, price and stock status
 - function to display correct message

#### Logic
 - function to restock inventory back to 10 per ingredient
 - function to update stock
 - function to check drink availablity
